A simple Form

Each form element requires three steps


  1. A row using form-group class

    2. < div class="form-group row" >
    .....
    < /div >

  2. A label for each form element that will use col-form-label
    3. < label for="firstname" >
      First Name
    < /div >

  3. Input element which will have same name as defined in label for and this will use bootstrap class of form-control
    4. < input type="input" class="form-control" name="firstname" />
